**Solar AI Advisor** is an intelligent web application that helps users predict solar energy generation potential for their rooftops using real NASA satellite data, and provides AI-powered guidance on Indian solar policies, subsidies, and installation procedures.

> 🌍 Solar calculations work worldwide. Policy guidance covers India (PM Surya Ghar, UPPCL, UPERC).

---

## 🌟 Features

### 🔆 Solar Energy Prediction
- **Real-time NASA Data**: Fetches actual solar irradiance from the NASA POWER API for any location worldwide
- **Accurate Generation Forecasts**: Calculates daily, weekly, monthly, and yearly solar energy generation
- **Financial Analysis**:
  - Installation cost estimate
  - PM Surya Ghar Yojana subsidy calculation
  - Net investment after subsidy
  - Monthly bill savings & payback period

### 🤖 AI-Powered Chatbot
- **RAG-based Knowledge System**: Uses Retrieval-Augmented Generation with a FAISS vector database built from official Indian solar policy PDFs
- **Answers questions about**:
  - Government subsidies and schemes (PM Surya Ghar)
  - Net metering policies
  - Installation eligibility criteria
  - Solar panel specifications
  - UPPCL / UPERC regulations
- **Quick-question buttons** for common queries
- **Document-grounded answers** — responds only from official policy documents

### 🗺️ Location Services
- **Interactive Map**: Click-to-select any location
- **GPS Integration**: Automatic location detection
- **City Presets**: One-click selection for major Indian cities
- **Global Coverage**: Works for any coordinates worldwide

### 📊 Visualisation
- **Interactive Charts**: Daily, weekly, monthly, and yearly generation bar/line graphs
- **Bill Comparison**: Side-by-side current vs post-solar electricity cost
- **Metric Cards**: Clear KPIs — system size, daily/monthly output, cost, subsidy, net investment

### 🎨 Modern UI/UX
- Responsive layout (desktop, tablet, mobile)
- Automatic dark/light theme detection
- WCAG AA accessible contrast ratios

## 🚀 Quick Start

### Prerequisites

| Requirement | Notes |
|-------------|-------|
| Python 3.9 – 3.13 | 3.11 recommended |
| Google Gemini API key | [Get it free here](https://aistudio.google.com/app/apikey) |

### Installation

#### Option A — Automated (Windows)
```bat
setup.bat
```

#### Option B — Automated (Mac / Linux)
```bash
chmod +x setup.sh && ./setup.sh
```

#### Option C — Manual step-by-step

```bash
# 1. Clone
git clone https://github.com/yourusername/solar_ai_advisor.git
cd solar_ai_advisor

# 2. Create & activate virtual environment
python -m venv venv
venv\Scripts\activate          # Windows
# source venv/bin/activate     # Mac / Linux

# 3. Install dependencies
pip install -r requirements.txt

# 4. Configure API key
copy .env.example .env         # Windows
# cp .env.example .env         # Mac / Linux
# Then edit .env and set:  GEMINI_API_KEY=your_key_here

# 5. Build the knowledge base (first time only)
python build_rag.py

# 6. Run the app
streamlit run app.py
```

Open **http://localhost:8501** in your browser.

## 🔧 Configuration

### NASA POWER API
No API key required. Free usage with rate limits.
- **Endpoint**: https://power.larc.nasa.gov/api/temporal/daily/point
- **Parameter**: ALLSKY_SFC_SW_DWN (surface solar irradiance, kWh/m²/day)
- **Fallback**: If the NASA API is unreachable, the app uses a seasonal estimate and shows a warning.

### Subsidy Rules — PM Surya Ghar Muft Bijli Yojana

| System Size | Central Financial Assistance |
|-------------|------------------------------|
| Up to 2 kW | ₹30,000 / kW |
| 3rd kW | ₹18,000 |
| Above 3 kW | Capped at ₹78,000 |

### UPPCL Domestic Tariff Slabs

| Units / Month | Rate (₹/unit) |
|---------------|---------------|
| 0 – 100 | 5.50 |
| 101 – 300 | 6.00 |
| 301 – 500 | 6.50 |
| Above 500 | 7.00 |

A ~10% fixed charge + electricity duty is added on top.
